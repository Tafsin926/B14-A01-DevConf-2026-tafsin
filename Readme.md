# DevConf 2026 Landing Page

A responsive landing page for **DevConf 2026**, built using HTML and CSS.

## Features

- Responsive navigation bar
- Hero banner with background image
- Speaker section
- Pricing section (Standard, Pro, Team)
- Benefits section
- Footer with social media icons

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Open Sans)
- Font Awesome Icons
